# java-cli-millbuild-postgresql-data-type-enum

## Description
Creates a small database table
called `dog`. Uses an enum user defined type.

## Tech stack
- java
- millbuild
  - 6.8.2

## Docker stack
- postgresql:alpine
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
